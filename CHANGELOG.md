# Changelog — MxM Tools

## 2.1.2 (current)

MxM Tools adds the following to the Musixmatch Studio task editor:

- 21 floating buttons: copy, save, save & send, find & replace, reset sync,
  lyrics comparison, Diffgenie, Song → Gem, review with Gem, contributor
  name/profile, open on the web, Google and streaming search (YouTube,
  Spotify, Apple Music, Amazon Music), and Typeform shortcuts.
- Configurable keyboard shortcuts: merge/split lines, wrap the selection in
  `( )` / `¿ ?` / `¡ !`, word counter, insert `#INSTRUMENTAL`, and Song → Gem.
- Highlighter for opened and keyword-matched songs.
- Original-lyrics capture on task open, and lyrics comparison
  (built-in view or Diffchecker).
- Gemini Gem integration: transcription, lyrics review and contributor
  message. The Gem instructions are shown on the in-app **Useful information**
  page.
- 26 UI languages, and full settings export/import.
